# XueqiuSpider

### 抓取雪球用户

1. 使用python requests和xpath与正则表达式抓取用户信息。    
2. 使用MongoDB储存用户信息，包括城市，性别，ID，关注数，粉丝数，讨论数。   
3. 使用redis做任务队列。   
4. 使用宽度优先算法
